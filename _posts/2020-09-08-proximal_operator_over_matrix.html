In this blog, we will show how to compute proximal operator of a constrained function. As an example we consider the function:

f(X,D):=∥Σ−X−D∥F2+IP(X,D),

where IP denotes the indicator function of the convex set

P={(X,D)∈Sn×Sn∣X⪰0,D=diag(d),d≥0,d∈Rn}.

Proximal operator proxγf for this function f at (X,D) is the optimal solution to the following convex optimization problem:

minimize∥Σ−X~−D~∥F2+12γ∥X~−X∥F2+12γ∥D~−D∥F2subject toX~⪰0D~=diag(d~)d~≥0,

where X~∈S+n, and d~∈R+n (i.e., D~=diag(d~)) are the optimization variables.

Now we solve this optimization problem using Julia. We will use the package Convex and COSMO, both open source Julia packages.
